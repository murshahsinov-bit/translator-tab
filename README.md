# 🌍 Табасаранский переводчик с GigaChat

Онлайн‑переводчик с русского на табасаранский язык и обратно с поддержкой автодополнения, тёмной темы, озвучивания и чата с GigaChat (ИИ от Сбера).

**Автор:** Шахсинов Мурад  
**Год:** 2026  
**Лицензия:** Все права защищены. Коммерческое использование платное.

---

## 📋 Возможности

- Перевод с русского на табасаранский и обратно.
- Словарь на 800+ слов.
- Автодополнение при вводе.
- Тёмная/светлая тема.
- Озвучивание переведённого текста.
- Чат с ИИ (GigaChat) по вопросам о языке.
- Адаптивный дизайн – работает на ПК, планшетах и телефонах.

## 🚀 Быстрый старт (для разработчиков)

1. Клонируйте репозиторий
git clone https://github.com/ваш-логин/translator-tab.git
cd translator-tab
3. Установите зависимости
bash
npm install
4. Получите ключ GigaChat
Зарегистрируйтесь в личном кабинете Сбера и получите ключ GIGACHAT_AUTH_KEY.

5. Создайте файл .env
Скопируйте .env.example в .env и вставьте свой ключ:

bash
cp .env.example .env
Отредактируйте .env:

text
GIGACHAT_AUTH_KEY=ваш_ключ_авторизации
PORT=3000
5. Запустите локально (для теста)
bash
node server.js
Откройте http://localhost:3000 в браузере.

6. Запустите в production (через PM2)
bash
npm install -g pm2
pm2 start server.js --name translator
pm2 save
pm2 startup
🛠 Полная установка на сервер (Ubuntu 22.04)
1. Подготовка сервера
bash
ssh root@ваш_ip_сервера
apt update && apt upgrade -y
apt install -y curl git nginx
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs
2. Клонируйте проект
bash
git clone https://github.com/ваш-логин/translator-tab.git
cd translator-tab
3. Настройка окружения
bash
cp .env.example .env
nano .env   # вставьте свой ключ
4. Установка и запуск
bash
npm install
npm install -g pm2
pm2 start server.js --name translator
pm2 save
pm2 startup
5. Настройка Nginx
bash
cat > /etc/nginx/sites-available/translator << 'EOF'
server {
    listen 80;
    server_name _;  # ваш домен, если есть
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
EOF

ln -s /etc/nginx/sites-available/translator /etc/nginx/sites-enabled/
nginx -t && systemctl reload nginx
ufw allow 80/tcp
6. Откройте сайт
http://ваш_ip_сервера

❓ Частые проблемы
GigaChat не отвечает – проверьте ключ в .env. Срок действия ключа – 30 дней, получите новый в кабинете Сбера.

Сайт не открывается на телефоне – вводите http://, а не https://. Если браузер автоматически переключает – используйте порт :8080.

Ошибка self-signed certificate – уже исправлена в коде. Если возникает, перезапустите PM2 с --update-env.
